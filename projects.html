Agile Management Tool

This project was designed to streamline team collaboration in agile development environments. Built with the Flask framework, the application features a front-end using HTML and CSS for clean, intuitive navigation and UI. The main backend logic is implemented in Python, with some JavaScript for dynamic elements and user interactions. Data is stored in an SQLite3 database to manage team progress, task assignments, and project status updates. Integrated with Jira, the tool enhances workflow transparency and productivity by allowing real-time tracking of team members and task stages.

James Dyson Competition

For the James Dyson Competition, our team focused on disaster response by developing a drone prototype designed for immediate post-disaster rescue. Using SolidWorks, we engineered a durable and versatile design that could assist in search and rescue operations. The 3D printing process enabled rapid prototyping, allowing us to test and refine the drone to perform effectively in emergency situations. This project was a blend of engineering and innovation, aiming to save lives and aid rescue efforts in challenging environments.

HVAC System (Electrical Engineering Project)

This HVAC project combined hardware and software engineering, using Arduino to control the system and Pymate to convert the code from C# to Python, making it compatible with other platforms. This project not only enhanced my skills in cross-language compatibility but also emphasized practical applications in electrical engineering. The end result was an efficient HVAC system with improved automation and user control, showcasing how software can drive hardware innovation in engineering solutions.

AI-Driven Pac-Man Game

In this project, I developed an AI-driven Pac-Man game utilizing object-oriented programming (OOP) principles in Python and integrated machine learning algorithms to adapt to player strategies. The AI analyzes player moves in real-time, adjusting the ghost behaviors and adding complexity to the gameplay. This project not only expanded my knowledge of machine learning but also demonstrated my ability to apply AI in creating interactive, intelligent gameplay experiences.
